¡Welcome to my cheat sheet!

This is a page that collects the most used tags in the programming language HTML and CSS.

HTML
h1 The "h1" tag is used to define the main header of a web page. It is recommended to use it only once per page and to ensure that the text is relevant and concise to improve accessibility and the structure of the content.
ul Se utiliza para crear una lista desordenada en HTML. Para su mejor práctica, se recomienda utilizar elementos "li" dentro de "ul" para cada ítem de la lista y estructurar el contenido de manera clara y legible. Además, se puede aplicar estilos CSS para personalizar el aspecto visual de la lista.
p La etiqueta "p" se utiliza para representar un párrafo en HTML. Es recomendable utilizarla para separar y estructurar el contenido de manera clara y legible. Se deben evitar párrafos excesivamente largos y se puede aplicar estilos CSS para ajustar el espaciado y la apariencia del texto.
